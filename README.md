# 📌 Nexo - Organizador de Tarefas e Projetos

## 📖 Sobre o Projeto

O **Nexo** é uma aplicação web inspirada no Trello, desenvolvida com **Angular** e **Tailwind CSS**, focada em organização de tarefas e projetos de forma simples, rápida e intuitiva.

A plataforma permite criar quadros de tarefas no estilo **Kanban**, adicionar cards, mover tarefas entre colunas utilizando **drag and drop**, editar informações e remover cards conforme necessário.

O objetivo principal do projeto é praticar conceitos modernos de desenvolvimento front-end utilizando Angular, componentização, gerenciamento de estado e estilização responsiva com Tailwind CSS.

---

# 🚀 Tecnologias Utilizadas

- Angular 17+
- Tailwind CSS
- TypeScript
- Angular CDK Drag and Drop
- RxJS

---

# 🎯 Funcionalidades

## ✅ Gerenciamento de Colunas
- Criar novas listas/colunas
- Editar título das colunas
- Excluir colunas

## ✅ Gerenciamento de Cards
- Criar tarefas/cards
- Editar conteúdo do card
- Excluir cards
- Adicionar descrição simples

## ✅ Drag and Drop
- Arrastar cards entre colunas
- Reordenar cards dentro da mesma coluna

## ✅ Interface
- Layout responsivo
- Design minimalista inspirado no Trello
- Feedback visual durante o arraste dos cards
